# Pokedex

## Tech Stack
- Vite + React SPA (JavaScript)
- Data from pokeapi.co
- No backend
- Deployed to GitHub Pages at `/<repo-name>/`

## Local Development

```bash
npm install
npm run dev      # Start dev server
npm run build    # Build for production
```

## Deployment
- Automatically deployed to GitHub Pages on push to main
- CI runs build checks on all pull requests

## Definition of Done
- Dependencies installed (package-lock.json committed)
- `npm run build` passes from a clean checkout
- Live page displays content without console errors

## Future Work
- All changes require branch + pull request
